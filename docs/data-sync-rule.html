数据同步相关

iHabit 的备份功能是将用户的数据直接保存到每个用户的 iCloud 空间下（用户独享的空间，依赖由苹果维护的云盘系统），而我们则不对这部分数据进行任何读取或者保存。

1. 如何开启 iCloud 备份功能？

请确保已为 iHabit 开启 iCloud 权限，需要检查开关：

  • 应用内部的设置页面 -> 数据备份 -> 开启自动备份；

  • 系统设置页面 -> Apple ID、iCloud、媒体与购买项目 -> iCloud -> 使用 ICLOUD 的 APP -> iHabit

  • 系统设置页面 -> Apple ID、iCloud、媒体与购买项目 -> iCloud -> iCloud 云盘。

2. 如何关闭 iCloud 备份功能？

应用内部的设置页面 -> 数据备份 -> 关闭自动备份；

3. iCloud 备份功能，是否会泄露隐私？

不会，隐私问题是我们非常看重的。iCloud 功能是由苹果公司提供且维护的，数据存储在每个用户的独立云盘空间内，不与其他用户的数据存放在一起。

另外，苹果公司在数据网络传输上有着非常严格的校验机制，传统的攻击手段例如中间人攻击是无法破坏数据的完整性的，我们有理由相信 iCloud 机制下数据传输的安全性。

4. iCloud 数据的传输是安全的，意味着数据 100% 不会出问题吗？

由于涉及网络的请求，本身具有失败的可能，另外受限网络环境的影响，可能出现数据无法及时同步的情况。如果数据无法及时同步，请采取以下措施尝试解决：

检测设备网络状态，并尝试切换成另一个网络进行尝试。
